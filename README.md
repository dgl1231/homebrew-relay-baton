# homebrew-relay-baton

Homebrew tap for [relay-baton](https://github.com/dgl1231/relay-baton).

```bash
brew tap dgl1231/relay-baton
brew install relay-baton
```

Installs the standalone CLI binary (macOS arm64 / Linux x64) from the
relay-baton GitHub Release. Formula source of truth lives in the main repo at
[`homebrew/relay-baton.rb`](https://github.com/dgl1231/relay-baton/blob/main/homebrew/relay-baton.rb).
